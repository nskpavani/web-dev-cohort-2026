# Cursor Landing Page Clone

A replica of the Cursor IDE landing page built with HTML and CSS.
<img width="1921" height="7092" alt="image" src="https://github.com/user-attachments/assets/9ac96e4d-8dc3-4c00-a692-0f924b4260b5" />

## Features

- Dark theme design matching Cursor's brand aesthetic
- Responsive layout with CSS Grid and Flexbox
- Multiple sections:
  - Header with navigation and CTA buttons
  - Hero section with tagline and download button
  - Features grid showcasing trusted platforms
  - Service sections highlighting Agent, Tab, and Ecosystem
  - User reviews/testimonials grid
  - Explore section with model access, codebase understanding, and enterprise features
  - Changelog section
  - Highlights/blog section
  - Footer with product, resources, company, legal, and connect links

## Files

- `landing-page.html` - Main HTML structure
- `styles.css` - All styling including responsive breakpoints
- `Cursor_Logo_1.png` - Cursor logo asset

## Setup

**1. Clone or download this repository
2. Open `landing-page.html` in your browser**

## Technologies Used

- Semantic HTML5
- CSS3 (Grid, Flexbox, Custom Properties)
- Responsive design with media queries
